# CareerUp – AI Career Platform

פלטפורמה חכמה לניהול חיפוש עבודה לסטודנטים ו-Juniors.

## 🚀 Demo
https://careerup1.netlify.app

## ✨ פיצ'רים
- 🤖 מאמן קריירה AI (Claude API)
- 📄 בונה קו"ח + התאמה למשרה
- 🎯 מאמן ראיונות עם אימון קולי
- 📊 דשבורד אנליטיקס אישי

## 🛠️ טכנולוגיות
HTML · CSS · JavaScript · Node.js · Express · Claude API (Anthropic)
